# Frontend

Next.js (App Router) frontend for the lead management app.

- `app/page.tsx` — public lead submission form
- `app/login/page.tsx` — attorney login
- `app/leads/page.tsx` — auth-guarded leads dashboard
- `lib/` — API client, auth token helpers

See the repo root [`README.md`](../README.md) and
[`docs/RUNNING_LOCALLY.md`](../docs/RUNNING_LOCALLY.md) for how to run this
alongside the backend.
